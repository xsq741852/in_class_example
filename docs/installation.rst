============
Installation
============


Stable release
--------------

To install in_class_example, run this command in your terminal:

.. code-block:: console

    $ pip install in_class_example

This is the preferred method to install in_class_example, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for in_class_example can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/xsq741852/in_class_example

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/xsq741852/in_class_example/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/xsq741852/in_class_example
.. _tarball: https://github.com/xsq741852/in_class_example/tarball/master
